# PopX App

A pixel-perfect React implementation of the PopX mobile app design.

## Tech Stack
- React 18
- React Router DOM v6
- Vite

## Pages
- `/` — Welcome screen with animated floating bubbles
- `/register` — Create account form
- `/login` — Sign in form
- `/account` — Account settings

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy on Vercel

Push to GitHub, then import repository in [Vercel](https://vercel.com). No extra config needed — Vite builds are auto-detected.
